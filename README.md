# eslint-config
Our standard lint configurations for TypeScript/JavaScript.

## Installation
```sh
npm install -D @maboroshi/eslint-config
```
or
```sh
yarn add -D @maboroshi/eslint-config
```

## Usage
Import the config to your `.eslintrc.json`
```json
{
  "extends": ["@maboroshi/eslint-config"]
}
```
